# MeuFoco - Identidade Visual (v1)

## Direcao Criativa
- Personalidade: clara, focada e acolhedora.
- Sensacao: produtividade sem rigidez, progresso diario com calma.
- Linguagem visual: base limpa, acentos energicos em verde petroleo, contrastes funcionais para dados.

## Nome e Assinatura
- Nome principal: `MeuFoco`.
- Uso recomendado em interface: sempre sem espaco (`MeuFoco`), com caixa alta no `M` e `F`.

## Paleta Principal

### Light
- Fundo base: `#F5FAF8`
- Fundo secundario: `#ECF5F2`
- Card: `#FFFFFF`
- Borda: `#D4E4DE`
- Texto principal: `#11211D`
- Texto secundario: `#3E5A53`
- Cor primaria (acao): `#0F8A78`
- Primaria hover: `#0C7566`
- Primaria suave: `#E0F6F1`

### Dark
- Fundo base: `#081512`
- Fundo secundario: `#10221E`
- Card: `#142B25`
- Borda: `#255247`
- Texto principal: `#E8F7F3`
- Texto secundario: `#A4CABE`
- Cor primaria (acao): `#33C0A5`
- Primaria hover: `#4FCFB6`
- Primaria suave: `rgba(51,192,165,0.18)`

## Cores de Apoio
- Risco/urgencia: `#DE5A5A` (light) / `#FF867F` (dark)
- Informacao: `#3F72E8` (light) / `#7CA0FF` (dark)
- Sucesso: `#2D9B6D` (light) / `#58D7A1` (dark)
- Destaque quente: `#C78F2A` (light) / `#F2C363` (dark)
- Destaque frio: `#0F9CAA` (light) / `#59DCE8` (dark)

## Tipografia
- Titulo e interface: `Sora`
- Texto de apoio: `Manrope`
- Numeros e dados: `JetBrains Mono`

## Regras de Uso
- CTA principal sempre na cor primaria.
- Estados ativos usam primaria com opacidade (10-20%), nunca blocos pesados.
- Cards e superficies permanecem neutros; o destaque visual vem de cor, nao de saturacao de fundo.
- Em dark mode, manter contraste alto entre texto e fundo para leitura rapida.
